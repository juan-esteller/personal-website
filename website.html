Website

In order to express my gratitude to the various resources that make this website possible and in the interest of (a slight abuse of) Kerchkoff's Principle, below are the technical details of this website:

  • I use the text editor Vim, usually inside of tmux.
  • This domain was purchased on and domain registration is provided by GoDaddy.
  • The source is hosted on GitHub.
  • SSL server certificates for the comforting padlock in your browser are from the Certificate Authority letsencrypt.
  • Web content is courteously served to you by nginx.
  • The website is hosted on a single Amazon Web Services Lightsail instance in Virginia. I would greatly appreciate not getting DOSed.
